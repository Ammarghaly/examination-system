Exam Management System
A comprehensive web-based platform designed to streamline the academic examination process, from user registration to detailed performance tracking.

Key Features
1. User Management
User Registration: An intuitive interface allowing new users to create accounts to access the system.
User Login: A secure authentication system to ensure data privacy and authorized access.

3. Admin & Management
Admin Panel: A dedicated dashboard for administrators to oversee and manage all components of the examination system.
Exams Dashboard: A centralized hub to view and manage available exams.
Update Exams: Provides the flexibility to modify, update, or edit existing exam data effortlessly.

4. Examination Process
Landing Page: A professional introductory page to welcome users to the project.
Examination Room: A distraction-free, interactive environment where students perform their tests.

5. Results & Tracking
Student Results: A quick-access feature to view grades and achievements.
Detailed Reports: Provides in-depth analytical reports on test performance to help students and instructors evaluate progress.

 Technical Stack & Architecture
The project utilizes a lightweight and efficient data management structure:
Data Storage: Uses db.json as a local database to store all system information (Users, Exams, Results).
API Layer: Uses Apis.js to handle all API calls and data communication between the frontend and the db.json file.
